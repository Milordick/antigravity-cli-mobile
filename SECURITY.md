# Security Policy

## Supported Versions

Only the latest released version of **antigravity-cli-mobile** is supported with security updates. If you find a security issue, please upgrade to the latest release to check if it has already been resolved.

| Version | Supported |
| --- | --- |
| Latest Release | ✅ Yes |
| All older versions | ❌ No |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do **NOT** open a public issue. Instead, report it privately via email:

- **Email:** parfenovacat26@gmail.com
- **Response Time:** We will acknowledge receipt of your report within 48 hours and provide a timeline for coordination and resolution.

- **Email:** parfenovacat26@gmail.com
